# Bianca Bezerra — Personal Portfolio


**Live Site:** [https://www.biabez.dev](https://www.biabez.dev)

Portfólio pessoal desenvolvido para apresentar minha identidade profissional, principais projetos, stack tecnológica e canais de contato em uma experiência moderna, responsiva e visualmente marcante.

A aplicação foi construída com foco em **performance**, **componentização**, **escalabilidade de interface** e **consistência visual**, refletindo minha abordagem no desenvolvimento frontend com tecnologias modernas do ecossistema React.

---

## Sobre o projeto

Este projeto funciona como minha vitrine profissional na web, reunindo em um único ambiente:

- apresentação pessoal
- projetos em destaque
- tecnologias que fazem parte da minha stack
- links para contato e networking profissional

Além de sua função institucional, o portfólio também representa uma aplicação prática de conceitos importantes como:

- organização de componentes reutilizáveis
- estruturação de aplicações com App Router
- separação de responsabilidades
- estilização modular
- construção de interfaces responsivas e modernas

---

Como executar localmente
Clone o repositório
git clone <url-do-repositorio>
Acesse a pasta do projeto
cd projetos-antigravity
Instale as dependências
npm install
Inicie o ambiente de desenvolvimento
npm run dev
Acesse no navegador
http://localhost:3000
Build de produção

Para gerar a build otimizada:

npm run build

Para iniciar a aplicação em produção:

npm start
Scripts disponíveis
npm run dev
npm run build
npm start
npm run lint
Descrição dos scripts

npm run dev → inicia a aplicação em ambiente de desenvolvimento

npm run build → gera a build otimizada para produção

npm start → executa a aplicação com a build de produção

npm run lint → roda a análise estática do projeto
